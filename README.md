# WizDesk – Team Management and Collaboration System

WizDesk is a web-based task management and collaboration platform designed to improve communication, coordination, and transparency within teams.

It allows team leaders to create and assign tasks while enabling members to track and update progress in real time.

## Features

- Team leader and member authentication
- Unique team code generation
- Leader approval system for members
- Task creation and assignment
- Real-time task progress tracking
- Team performance analytics
- Transparent team dashboards
- Active and completed task monitoring

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

## System Workflow

1. Team leader creates a team
2. Unique team code is generated
3. Members join using the team code
4. Leader approves member requests
5. Tasks are assigned and tracked
6. Members update progress in real time
